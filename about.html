What is Java?

Java is an advanced software development project for network devices and embedded systems. It is one of the most popular programming languages in the world. It can be used for apps, games, Artificial intelligence (AI), interent of things (IoT), and much more.

Some Java Basics:

Data types

Variables are the containers for storing data, here are the ones I've used.

  • String- stores text by using double quotes, such as "Hello".
  • int- stores whole numbers, without decimals.
  • double/float- stores floating point numbers, with decimals.
  • char- stores single characters by using single quotes, such as 'B'
  • boolean- stores value with either true or false.

Methods

A method is a block of code that run when it's called. They are used to perform certain actions and can also be known as functions.

A method consists of:

  • Modifier access either private or public.
  • Return type which returns the value.
  • Method name
  • Parameters
  • Method body

Constructors

In java a constructor is used to initialize objects and is invoked when an object of the class is created. Unlike a method the constrcutor has the same name as the class and does not have a return type.

In Java, constructors can be divided into 3 types:

  1. No-Arg Constructor
  2. Parameterized Constructor
  3. Default Constructor
